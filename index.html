DNA Sequence Prediction using Deep Learning

Built a deep learning pipeline to predict the next 3 nucleotides in human DNA (GRCh38 genome) bases of preceding context, framing genomic sequence prediction as an autoregressive language-modeling task.

Deep Learning CNN + Transformer CNN + BiLSTM Genomics

Designed and trained two hybrid neural architectures — CNN + Transformer (self-attention) and CNN + BiLSTM — achieving 77.84 percent test accuracy per base against a 25 percent random baseline.

Engineered a leak-proof evaluation protocol by splitting train/validation/test sets at the whole-chromosome level (rather than random sampling) to eliminate sequence overlap between splits.

Conducted error analysis: measured accuracy separately for repetitive vs. unique genomic regions, per-predicted-position accuracy, and k-mer overlap between train/test sets to distinguish genuine pattern learning from repeat memorization.
